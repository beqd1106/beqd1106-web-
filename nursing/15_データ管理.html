Google Sheets 連携設定

初回セットアップ手順

  1. 1Google Cloud Console でプロジェクト作成 → Google Sheets API を有効化
  2. 2「IAM と管理」→「サービスアカウント」でアカウント作成 → JSONキーをダウンロード
  3. 3Render.com の「Environment」→ 変数名 GOOGLE_CREDENTIALS_JSON に JSONの中身をそのまま貼り付け
  4. 4Google Sheetsで新規スプレッドシートを作成 → サービスアカウントのメールアドレスを「編集者」として共有
  5. 5スプレッドシートのURL（spreadsheets/d/ここがID/edit）のIDを下に入力
このIDはブラウザのLocalStorageに保存されます（サーバーには送信されません）
